How to set up the sheet (one time, ~2 min)
  1. Create a blank Google Sheet (the tabs are created automatically).
  2. In it: Extensions ▸ Apps Script. Delete any code, paste the contents of apps-script/Code.gs from this project, and Save.
  3. Deploy ▸ New deployment ▸ Web app. Set Execute as: Me and Who has access: Anyone. Authorise when asked.
  4. Copy the /exec URL, paste it above, hit Test connection, then Save settings.

Your data then lives in your Google account and syncs to any device that opens this page with the same URL.
